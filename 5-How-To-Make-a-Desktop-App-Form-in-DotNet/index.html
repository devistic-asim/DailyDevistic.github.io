Let's Get Started...

Step 1: Create a New Project

Follow these steps to create a new desktop application project using Visual Studio 2019:

  1. Open Visual Studio 2019.
  2. Click on "Create a new project."
  3. Search for "Windows Forms App (.NET Framework)."
  4. Choose a name and location for your project and click "Create."

Step 2: Design the Form

Design the user interface for your desktop app form:

  1. In the Solution Explorer, right-click on your project and select "Add" > "Windows Form."
  2. Double-click the newly added form (Form1.cs) to open the designer.
  3. Drag and drop the required controls onto the form: 2 TextBoxes, 4 ComboBoxes, and a Button.
  4. Set the names and properties of the controls appropriately.

Step 3: Writing Code

Write C# code to handle form actions:


    using System;
    using System.IO;
    using System.Windows.Forms;
    
    namespace FileCopyApp
    {
        public partial class Form1 : Form
        {
            public Form1()
            {
                InitializeComponent();
            }
    
            private void btnSubmit_Click(object sender, EventArgs e)
            {
                string sourceDirectory = txtSourceDirectory.Text;
                string targetDirectory = txtTargetDirectory.Text;
                string fileName = txtFileName.Text;
    
                string sourceFilePath = Path.Combine(sourceDirectory, fileName);
                string targetFilePath = Path.Combine(targetDirectory, fileName);
    
                try
                {
                    File.Copy(sourceFilePath, targetFilePath, true);
                    MessageBox.Show("File copied successfully!");
                }
                catch (Exception ex)
                {
                    MessageBox.Show("An error occurred: " + ex.Message);
                }
            }
        }
    }

Step 4: Configure ComboBoxes

Populate the ComboBoxes with the required values in the form's constructor or "Load" event.

Step 5: Build and Run

Build and run your application to test it:

  1. Build the solution (Ctrl + Shift + B).
  2. Run the application (F5).

Step 6: Test the Application

Test the functionality of your application by entering input and clicking the "Submit" Button.

Remember to customize the code according to your requirements and add appropriate error handling. Thank you
